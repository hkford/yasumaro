## 失敗時の対処

テストが jsdom アノテーションを追加しても失敗する場合:

1. **エラー内容を確認:** `npx vitest run <file> --reporter=verbose 2>&1 | grep "Error\|FAIL" | head -30`
2. **よくある原因:**
   - `chrome.storage` が未モック → `vitest.setup.ts` の chrome mock を確認
   - `fetch` が未モック → `vi.mock('fetch', ...)` を追加
   - DOM 要素が期待通り存在しない → `document.body.innerHTML` のセットアップを確認
3. **修正不可能な場合:** そのファイルを再 exclude してコメントで理由を記録する
