# 詳細


## 1. WXTへ完全移行

- **目標:** 単一コードベースからChrome/Firefox/Edge 全対応ビルド生成
- **指標:** `wxt build -b chrome|firefox|edge` がすべて成功
- **対応:**
  - WXTビルド確認と残留スクリプトの排除
  - クロスブラウザ動作確認


## 2. TypeScript厳格化

- **目標:** 型安全性の大幅向上
- **指標:** 
  - `tsc --strict` エラー0
  - カバレッジ 80%以上
- **対応:**
  - `tsconfig.json` の `strict: true` 完全適用
  - `any` 型の廃止、`unknown` による置換
  - 未使用変数の削除


## 3. Svelte + Tailwind導入

- **目標:** UI保守性の向上とCSS干渉問題の解決
- **指標:**
  - UIコンポーネントの再利用性 50%向上
  - 新規UI実装時におけるCSS干渉報告 0件
- **対応:**
  - Popup UIのSvelteコンポーネント化
  - Tailwindユーティリティクラスの標準化
  - 既存vanilla UIの漸進的移行（優先度: Popup > Dashboard > Content Script）


## 4. AIプロバイダー抽象化

- **目標:** 新規AIプロバイダー追加のPlug & Play化
- **指標:** 新規プロバイダー追加時に必要な変更箇所を5ファイル以内に
- **対応:**
  - `ProviderStrategy` インターフェース整備
  - プロバイダー追加時のチェックリスト作成
  - ドキュメントへのプロバイダー追加ガイド追加


## 5. E2Eテスト拡充

- **目標:** 自動テストによる品質保証の強化
- **指標:**
  - Playwrightシナリオ 30%以上増
  - ユニットテスト + E2E 合算カバレッジ 70%以上
- **対応:**
  - 主要ユーザーフロー（保存、要約、履歴表示）のPlaywright化
  - クロスブラウザテストの追加（Playwright設定追加済み、要動作確認）
  - CIへのE2Eテスト組み込み


## 6. エラーコード体系

- **目標:** ユーザー体験向上のエラー処理
- **指標:**
  - 全エラーに統一的なエラーコード付与
  - recoverable/unrecoverable 分類完了
  - ユーザーフレンドリーエラーメッセージ 100%カバー
- **対応:**
  - `docs/ERROR_CODES.md` 拡張
  - エラーコード分類（APIエラー、ネットワークエラー、ユーザー操作エラー）
  - 各エラーコードに対するリカバリー手段の文書化


## 7. ドキュメント整備

- **目標:** 開発者フレンドリーなドキュメント体制
- **指標:**
  - Typedocカバレッジ 100%（公開API）
  - ADRドキュメント完全化
- **対応:**
  - JSDoc → Typedoc移行
  - ドメイン層設計書（Infrastructure/Domain/UIレイヤーの分離）作成


## 8. CI/CD整備

- **目標:** リリースフロー自動化
- **指標:**
  - セマンティックバージョニング 100%遵守
  - 手動リリース作業 0件
- **対応:**
  - GitHub Actions によるリリースワークフロー構築


## 9. アクセシビリティ

- **目標:** Lighthouse Accessibility 95%以上
- **対応:**
  - aria-labelの完備
  - フォーカス管理確認
  - カラーコントラストチェック


# 備考

- **Safari対応について**
  - Apple Developer Program（年間99ドル）加入が必要のため、今回は見送り
  - 将来的に余裕があれば再検討

- **パフォーマンステストについて**
  - ベンチマークやプロバイダー別SLA監視は難易度が高いため見送り
  - 実需発生時に再検討

- **キーボード操作対応について**
  - 本拡張は「じっくり読んだWebページを記録」が本質
  - ユーザーがスクロールして読んだページを保存するため、キーボード操作は必須でない
  - ADRとして記録：「キーボード操作対応は実施しない」

- **スクリーンリーダー対応について**
  - 実装難易度とユーザー層を考慮し、今回は見送り
